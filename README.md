# FlyNow

A digital flyers and coupon platform built using React, TypeScript, Vite, and Supabase.

## Getting Started

This project allows users to browse digital flyers, discover the latest offers, save coupons to a digital wallet, and earn rewards through interactive features such as Spin & Win, Scratch Cards, Daily Check-In, and Refer & Earn.

### Features

- Browse offers by category
- Search and filter coupons
- Digital Coupon Wallet
- QR Code coupon redemption
- Reward Wallet
- Spin & Win
- Scratch Card
- Daily Check-In
- Refer & Earn

### Tech Stack

- React
- TypeScript
- Vite
- Supabase
- PostgreSQL
